🎯 First-Round Combo
Ideal opening (target surprised):
1. Hide / Sneak into position before combat
2. Attack with Action (Shortsword) → auto-crit (Assassinate) + Sneak Attack 2d6 → all dice doubled
3. Attack with Bonus Action (off-hand Shortsword) → another crit if lucky (19-20), + Sneak Attack if no other attack qualified
4. If needed: Action Surge → two more attacks this turn
5. Total potential: 4 attacks in one round on a surprised target
